# image2.0用 サイト全体モック生成プロンプト

以下のプロンプトをimage2.0に渡し、サイト全体のデザインモックを作成する。

---

恐竜タイプ診断サイトのUI/UXデザインモックを作成してください。

目的:
ユーザーが24問の性格診断に答えると、5軸スコアから16種類の恐竜タイプに分類されるWebサイトです。可愛いとカッコイイを両立し、スマホで見たときに完成度が高いデザインにしてください。

世界観:
古代、恐竜、冒険、DNA、琥珀、化石、ジャングル、夜のネオン、スマホゲーム風の高品質UI。

デザイン方針:
- スマホファースト
- 縦長画面
- 可愛いマスコット恐竜と、迫力あるリアル寄り恐竜の両方を活かす
- 背景は暗めで高級感、アクセントに琥珀色、エメラルド、ターコイズ、赤を使う
- UIは丸みがあり、ボタンは押したくなる質感
- 文字は読みやすく、診断サイトとして信頼感がある
- 既存のMBTI/16Personalities風にはしない

作成してほしい画面:
1. トップページ
   - サイト名
   - キャッチコピー
   - リアル恐竜とマスコット恐竜が共存するメインビジュアル
   - 診断開始ボタン
   - 「24問であなたの恐竜タイプを診断」などの説明

2. 診断ページ
   - 質問文
   - 5つの選択肢カード
   - 進捗バー
   - 戻るボタン
   - 背景にDNA/化石/葉の装飾

3. 結果ページ
   - タイプ名
   - 恐竜名
   - キャッチコピー
   - リアル恐竜イラスト
   - マスコット恐竜イラスト
   - 5軸レーダーチャート
   - 強み、注意ポイント、相性
   - 結果画像保存ボタン
   - もう一度診断ボタン

4. 図鑑ページ
   - 16タイプのカード一覧
   - 各カードにマスコット恐竜、タイプ名、短い説明

出力形式:
- 1枚の総合モックでもよいが、可能なら各画面ごとに分ける。
- 実装者がHTML/CSSに落とし込みやすいように、余白、カード構造、ボタン、配色が分かるようにする。
- 文字は日本語で自然に表示する。
