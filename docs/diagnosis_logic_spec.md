# 診断ロジック仕様

## 1. 基本方針
MBTIのコピーではなく、独自の5軸スコアリングで判定する。

## 2. 診断軸
- aggression: 攻撃性 / 決断力 / 主導性
- cooperation: 協調性 / 共感 / チーム志向
- curiosity: 探究心 / 分析 / 学習意欲
- caution: 慎重さ / 安定志向 / リスク管理
- speed: スピード感 / 行動力 / 反応速度

## 3. 設問数
推奨: 24問
- 短すぎると診断の信頼感が弱い。
- 長すぎると離脱しやすい。
- 24問なら、5軸をおおむね均等に測れる。

## 4. 回答方式
各設問は5択。
各選択肢がいずれかの軸に +2、または複数軸に +1/+1 を加点する。

例:
```json
{
  "text": "突然ピンチになったら、まずどう動く？",
  "options": [
    { "label": "正面から突破する", "scores": { "aggression": 2 } },
    { "label": "仲間と相談する", "scores": { "cooperation": 2 } },
    { "label": "原因を分析する", "scores": { "curiosity": 2 } },
    { "label": "安全な道を探す", "scores": { "caution": 2 } },
    { "label": "すぐに動いて状況を見る", "scores": { "speed": 2 } }
  ]
}
```

## 5. 判定方式
### 5.1 基本
1. 5軸の合計点を算出する。
2. 最大軸をprimaryAxis、2番目をsecondaryAxisとする。
3. primaryAxis + secondaryAxis の組み合わせから候補タイプを決める。
4. 同点時は、回答の終盤10問の得点を優先してタイブレークする。
5. それでも同点なら、タイプ定義のpriorityで決定する。

### 5.2 16タイプ化
5軸の単純な2軸組み合わせだけでは10通りにしかならないため、16タイプは以下の要素を組み合わせる。
- primaryAxis
- secondaryAxis
- 極端値: 1軸が突出して高い場合
- バランス型: 上位3軸の差が小さい場合
- 低スコア補正: 極端に低い軸がある場合

## 6. 診断の納得感を上げるためのルール
- 結果説明では「あなたは絶対こう」と断定しすぎない。
- 「〜しやすい」「〜な場面で力を発揮する」のように表現する。
- 弱みは否定ではなく、注意ポイントとして書く。
- 結果には上位2軸とレーダーチャートを表示し、判定理由を見せる。

## 7. 免責
この診断は医療・心理検査ではなく、独自ロジックによるエンタメ性格診断であることを明記する。
