# 結果カード仕様

## 1. 目的
ユーザーが診断結果を画像として保存したくなるカードを生成する。

## 2. 保存画像サイズ
- 推奨: 1080x1920px
- スマホ壁紙・SNS投稿に流用しやすい縦長比率。

## 3. 表示要素
- サイト名/ロゴ
- タイプ名
- 恐竜名
- キャッチコピー
- マスコット恐竜イラスト
- リアル恐竜イラストの小さめ背景またはシルエット
- 5軸スコア
- レーダーチャート
- 強み3つ
- 注意ポイント1つ
- 診断日

## 4. 実装方針
- HTML要素をカードとして組み、html2canvas等で画像化する。
- 依存ライブラリを使う場合はライセンスを確認し、GitHub Pagesで動くものに限定する。
- 画像保存に失敗した場合は、カード部分を長押し/スクリーンショットする案内を出す。
